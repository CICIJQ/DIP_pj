# Released Results

This directory contains a compact, Git-friendly export of key experiment
artifacts produced in this project.

Included here:

- RDNet ablation tables and aggregate summaries
- per-benchmark summary files for the fair `RDNet + NAFNet Refiner` run
- per-benchmark summary files for the fair `RDNet + Learned Confidence Gate` run
- training metadata for the two fair RDNet refiner runs
- self-collected 5-image comparison outputs used in the final analysis

Not included here:

- full per-image benchmark outputs for every dataset
- checkpoints and model weights
- raw datasets
- tensorboard logs and large intermediate artifacts

Refer to the main `ERRNet/` documentation for code details and to these
released files for the main result numbers discussed in the project notes.
