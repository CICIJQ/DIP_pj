# DIP_pj

This repository is a code-only export of the local `dip_pj` workspace.

Included subprojects:

- `ERRNet/`: ERRNet-based reflection removal experiments, ablations, and RDNet refinement code.
- `DSRNet/`: DSRNet-related training and evaluation code used in the same project workspace.

Excluded from this GitHub export:

- datasets and raw data
- checkpoints and pretrained weights
- generated results and logs
- local virtual environments and cache files

See each subproject's own `README.md` for setup and usage details.
